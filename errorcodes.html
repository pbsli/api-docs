Survey API Error Codes

Updated on February 20, 2013

When a response to an API request returns an HTTP status code other than 200, it will be accompanied by a special error response format:

Example Response

{
    errorCode: "BadParameter",
    message: "A URL or JSON parameter you supplied was improperly formatted or identifies an invalid resource."
}

The above error response would accompany an HTTP status code of 400, indicating that a parameter provided in the request was invalid.

The following table lists the error codes that may be returned by the system:

HTTP Status Code Error Code Message
403 AccessDenied Your request specified a respondent, survey or response (or some combination thereof) that you are not permitted to access.
403 AccountProblem There is a problem with your API account that prevents the operation from completing successfully. Please contact support.
400 BadDigest The Content-MD5 you specified does not match what we received.
400 BadParameter A URL or JSON parameter you supplied was improperly formatted or identifies an invalid resource.
404 EndpointNotFound There is no API endpoint defined for the URL you requested.
500 InternalError We encountered an internal error. Please try again.
400 InvalidAcceptList The server does not know how to respond in any of the formats specified in your request's Accept header.
400 InvalidContentType You specified a content type that is not supported.
403 InvalidPublicKey The public key token provided does not exist in our records.
403 InvalidSignatureIdentifier The Authorization signature identifier you provided is not known by the system.
400 MalformedHeaderValue An HTTP header value was malformed.
400 MalformedJson The JSON document submitted with your request was malformed.
400 MissingHeader The request was missing a required HTTP header. Please review the documentation for a list headers required for API requests.
400 MissingRequestBody The request used an HTTP verb that requires a content body, but no content body was sent.
400 MissingSecurityHeader Your request was missing a required header.
400 RequestTimeTooSkewed The difference between the request time and the server's time is too large.
503 ServiceNotEnabled The API service is not currently accepting requests.
403 SignatureDoesNotMatch The request signature we calculated does not match the signature you provided.
503 SlowDown Please reduce your request rate.
